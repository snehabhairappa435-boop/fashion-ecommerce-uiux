
# Fashion E-Commerce UI/UX Design

## Project Overview

A modern Fashion E-Commerce UI/UX design created in Figma for both Mobile Application and Responsive Website.

The project focuses on creating a clean, minimal, and user-friendly shopping experience with intuitive navigation, modern layouts, and responsive design principles.

---

## Features

### Mobile Application

- User Onboarding
- Login & Registration
- Home Screen
- Product Categories
- Product Listing
- Product Details
- Shopping Cart
- Checkout
- Order Success Screen

### Website

- Landing Page
- Shop Page
- Product Details
- Shopping Cart
- Wishlist
- Checkout
- Testimonials
- Newsletter Subscription
